使用methods中的函数来计算属性，则每次使用该属性会重新调用该函数，浪费性能。

与v-show的区别

  • v-if会从根本上直接删除或增加DOM ( 审查元素出现或消失 )
  • v-show只是给元素添加或删除一个行内样式display:none( 审查元素一直存在 )
  • 需要频繁切换的场景使用v-show节省性能
  • 一次性或者少量切换场景使用v-if
